Chapter 1 — Solving Linear Equations

Goal: isolate the variable. Use inverse operations in reverse PEMDAS order. Distribute first if needed; combine like terms; move variables to one side; solve.

Problem 1.1

Solve: 3(x − 4) + 2x = 17

Show solution
  1. Distribute: 3x − 12 + 2x = 17.
  2. Combine: 5x − 12 = 17.
  3. Add 12: 5x = 29.
  4. Divide by 5: x = 29/5 (or 5.8).

Answer: x = 29/5

Problem 1.2

Solve: 4x − 7 = 2x + 9

Show solution
  1. Subtract 2x: 2x − 7 = 9.
  2. Add 7: 2x = 16.
  3. Divide: x = 8.

Answer: x = 8

Problem 1.3

Solve: (x/3) + 5 = 11

Show solution
  1. Subtract 5: x/3 = 6.
  2. Multiply by 3: x = 18.

Answer: x = 18

Problem 1.4

Solve the literal equation for y: 3x + 4y = 12

Show solution
  1. Subtract 3x: 4y = 12 − 3x.
  2. Divide by 4: y = (12 − 3x)/4 = 3 − (3/4)x.

Answer: y = 3 − (3/4)x

Problem 1.5

Solve: 2(3x − 5) − 4(x + 1) = 6

Show solution
  1. Distribute: 6x − 10 − 4x − 4 = 6.
  2. Combine: 2x − 14 = 6.
  3. Add 14: 2x = 20.
  4. Divide: x = 10.

Answer: x = 10

Problem 1.6

Word problem: A taxi charges a $3 flat fee plus $2 per mile. If your total ride was $19, how many miles did you travel?

Show solution
  1. Let m = miles. Equation: 3 + 2m = 19.
  2. Subtract 3: 2m = 16.
  3. Divide: m = 8.

Answer: 8 miles

Chapter 2 — Inequalities

Solve like equations BUT flip the inequality sign whenever you multiply or divide both sides by a negative number. For compound 'and' (between), solve all parts together; for 'or', solutions can satisfy either.

Problem 2.1

Solve and graph the solution: −3x + 5 ≤ 14

Show solution
  1. Subtract 5: −3x ≤ 9.
  2. Divide by −3 (flip!): x ≥ −3.

Answer: x ≥ −3 (closed dot at −3, arrow right)

Problem 2.2

Solve: 2(x − 3) > x + 4

Show solution
  1. Distribute: 2x − 6 > x + 4.
  2. Subtract x: x − 6 > 4.
  3. Add 6: x > 10.

Answer: x > 10

Problem 2.3

Solve the compound inequality: −1 ≤ 2x + 3 < 9

Show solution
  1. Subtract 3 from each part: −4 ≤ 2x < 6.
  2. Divide each by 2: −2 ≤ x < 3.

Answer: −2 ≤ x < 3

Problem 2.4

Solve the compound inequality (or): x − 4 < −2 OR 3x ≥ 15

Show solution
  1. Left: x − 4 < −2 → x < 2.
  2. Right: 3x ≥ 15 → x ≥ 5.

Answer: x < 2 OR x ≥ 5

Problem 2.5

Solve the absolute-value inequality: |2x − 1| < 7

Show solution
  1. Rewrite: −7 < 2x − 1 < 7.
  2. Add 1 to each: −6 < 2x < 8.
  3. Divide by 2: −3 < x < 4.

Answer: −3 < x < 4

Chapter 3 — Functions & Function Notation

A relation is a function if every input has exactly one output (vertical line test on a graph). Use f(x) notation: f(3) means 'plug 3 in for x.' Domain = set of allowed inputs; range = set of outputs.

Problem 3.1

If f(x) = 2x2 − 5x + 1, find f(3) and f(−2).

Show solution
  1. f(3) = 2(9) − 5(3) + 1 = 18 − 15 + 1 = 4.
  2. f(−2) = 2(4) − 5(−2) + 1 = 8 + 10 + 1 = 19.

Answer: f(3) = 4; f(−2) = 19

Problem 3.2

If f(x) = 3x − 7 and g(x) = x2 + 2, find f(g(2)).

Show solution
  1. First g(2) = 4 + 2 = 6.
  2. Then f(6) = 3(6) − 7 = 11.

Answer: f(g(2)) = 11

Problem 3.3

Determine whether {(1, 2), (3, 5), (1, 7), (4, 9)} is a function. Explain.

Show solution
  1. x = 1 maps to two different y-values (2 and 7). Not a function.

Answer: Not a function: input 1 has two outputs.

Problem 3.4

If f(x) = 2x + 5, find f(x + 3) in simplified form.

Show solution
  1. Replace x with (x + 3): f(x + 3) = 2(x + 3) + 5 = 2x + 6 + 5 = 2x + 11.

Answer: f(x + 3) = 2x + 11

Problem 3.5

State the domain and range of the function shown by the points: (−2, 5), (0, 1), (3, −4), (4, 7).

Show solution
  1. Domain = set of x-values: {−2, 0, 3, 4}.
  2. Range = set of y-values: {−4, 1, 5, 7}.

Answer: Domain {−2, 0, 3, 4}; Range {−4, 1, 5, 7}.

Chapter 4 — Linear Functions & Graphing

Slope m = (y2 − y1)/(x2 − x1). Slope-intercept form y = mx + b. Point-slope form y − y1 = m(x − x1). Parallel lines have equal slopes; perpendicular slopes multiply to −1.

Problem 4.1

Find the slope between (−2, 5) and (4, −7).

Show solution
  1. m = (y2 − y1)/(x2 − x1) = (−7 − 5)/(4 − (−2)) = −12/6.

Answer: m = −2

Problem 4.2

Write the equation of the line through (3, −1) with slope 2 in slope-intercept form.

Show solution
  1. y = mx + b → −1 = 2(3) + b → −1 = 6 + b → b = −7.

Answer: y = 2x − 7

Problem 4.3

Write the equation of the line through (1, 4) and (5, −4) in slope-intercept form.

Show solution
  1. Slope: m = (−4 − 4)/(5 − 1) = −8/4 = −2.
  2. Use (1, 4): 4 = −2(1) + b → b = 6.

Answer: y = −2x + 6

Problem 4.4

Write the equation of the line parallel to y = (1/3)x − 2 that passes through (6, 5).

Show solution
  1. Parallel → same slope, m = 1/3.
  2. Use point (6, 5): 5 = (1/3)(6) + b → 5 = 2 + b → b = 3.

Answer: y = (1/3)x + 3

Problem 4.5

Write the equation of the line perpendicular to y = (1/2)x + 4 passing through (−2, 3).

Show solution
  1. Perpendicular slope is the negative reciprocal: m = −2.
  2. Use point (−2, 3): 3 = −2(−2) + b → 3 = 4 + b → b = −1.

Answer: y = −2x − 1

Problem 4.6

Convert 4x − 3y = 12 to slope-intercept form. State slope and y-intercept.

Show solution
  1. Subtract 4x: −3y = −4x + 12.
  2. Divide by −3: y = (4/3)x − 4.

Answer: Slope 4/3; y-intercept −4.

Chapter 5 — Systems of Equations

A system has TWO equations with TWO unknowns. Substitution: solve one for a variable and plug in. Elimination: add or subtract the equations (after scaling) to cancel a variable. One solution = lines intersect; no solution = parallel; infinitely many = same line.

Problem 5.1

Solve by substitution: y = 2x + 1 and 3x + y = 16

Show solution
  1. Substitute first into second: 3x + (2x + 1) = 16.
  2. 5x + 1 = 16 → 5x = 15 → x = 3.
  3. y = 2(3) + 1 = 7.

Answer: (3, 7)

Problem 5.2

Solve by elimination: 2x + 3y = 12 and 2x − y = 4

Show solution
  1. Subtract second from first: (2x + 3y) − (2x − y) = 12 − 4.
  2. 4y = 8 → y = 2.
  3. Substitute: 2x − 2 = 4 → x = 3.

Answer: (3, 2)

Problem 5.3

Solve: 3x + 2y = 11 and 5x − 4y = 11

Show solution
  1. Multiply first by 2: 6x + 4y = 22.
  2. Add to second: 11x = 33 → x = 3.
  3. Substitute: 3(3) + 2y = 11 → 9 + 2y = 11 → y = 1.

Answer: (3, 1)

Problem 5.4

Word problem: A coffee shop sells small drinks for $3 and large drinks for $5. On Monday they sold 80 drinks for $310. How many of each were sold?

Show solution
  1. Let s = small, L = large. s + L = 80; 3s + 5L = 310.
  2. From first: s = 80 − L. Substitute: 3(80 − L) + 5L = 310.
  3. 240 − 3L + 5L = 310 → 2L = 70 → L = 35.
  4. s = 80 − 35 = 45.

Answer: 45 small, 35 large

Problem 5.5

How many solutions? 2x − 3y = 6 and −4x + 6y = −12

Show solution
  1. Multiply first by −2: −4x + 6y = −12. Same as second.
  2. The two equations are equivalent → same line.

Answer: Infinitely many solutions

Chapter 6 — Exponents & Exponential Functions

Exponent rules: same base → add exponents (multiplication) or subtract (division). Power of a power: multiply exponents. Negative exponents flip; zero exponent = 1. Exponential growth: y = a(1 + r)t. Decay: y = a(1 − r)t.

Problem 6.1

Simplify: (3x2y3)(4x5y)

Show solution
  1. Multiply coefficients: 3·4 = 12.
  2. Add exponents on like bases: x2·x5 = x7; y3·y1 = y4.

Answer: 12x7y4

Problem 6.2

Simplify: (2x3)4

Show solution
  1. Power of a product: 24 · (x3)4.
  2. = 16 · x12 = 16x12.

Answer: 16x12

Problem 6.3

Simplify: (x5y2) / (x2y−1)

Show solution
  1. Subtract exponents: x(5−2) = x3; y(2−(−1)) = y3.

Answer: x3y3

Problem 6.4

Write 0.00045 in scientific notation.

Show solution
  1. Move decimal 4 places to the right: 4.5. The exponent is −4.

Answer: 4.5 × 10−4

Problem 6.5

Exponential decay: A car's value depreciates 12% per year. It originally costs $25,000. Write the function and find its value after 5 years.

Show solution
  1. Decay: y = 25000(1 − 0.12)t = 25000(0.88)t.
  2. At t = 5: 25000(0.88)5 ≈ 25000(0.5277) ≈ 13,193.

Answer: y = 25000(0.88)t; ≈ $13,193 after 5 years.

Problem 6.6

Find the next term in the geometric sequence: 3, 12, 48, 192, …

Show solution
  1. Common ratio r = 12/3 = 4.
  2. Next term: 192 · 4 = 768.

Answer: 768

Chapter 7 — Polynomials & Factoring

Always factor out the GCF first. Patterns: difference of squares a2−b2 = (a+b)(a−b); perfect square trinomial a2±2ab+b2 = (a±b)2; trinomial x2+bx+c → find two numbers product c, sum b. For ax2+bx+c with a > 1, use AC method.

Problem 7.1

Multiply: (2x − 3)(x + 5)

Show solution
  1. FOIL: 2x·x + 2x·5 − 3·x − 3·5 = 2x2 + 10x − 3x − 15.
  2. Combine: 2x2 + 7x − 15.

Answer: 2x2 + 7x − 15

Problem 7.2

Add: (3x2 − 4x + 7) + (x2 + 2x − 5)

Show solution
  1. Combine like terms: (3+1)x2 + (−4+2)x + (7−5).

Answer: 4x2 − 2x + 2

Problem 7.3

Factor: x2 + 8x + 15

Show solution
  1. Need two numbers with product 15, sum 8 → 3 and 5.

Answer: (x + 3)(x + 5)

Problem 7.4

Factor: x2 − 5x − 24

Show solution
  1. Two numbers with product −24, sum −5 → −8 and 3.

Answer: (x − 8)(x + 3)

Problem 7.5

Factor: 6x2 + 11x − 10

Show solution
  1. AC = 6 · (−10) = −60. Need numbers with product −60, sum 11 → 15 and −4.
  2. Split middle: 6x2 + 15x − 4x − 10.
  3. Group: 3x(2x + 5) − 2(2x + 5) = (3x − 2)(2x + 5).

Answer: (3x − 2)(2x + 5)

Problem 7.6

Factor: 9x2 − 49

Show solution
  1. Difference of squares: (3x)2 − (7)2.

Answer: (3x + 7)(3x − 7)

Problem 7.7

Factor completely: 3x3 − 12x

Show solution
  1. GCF 3x: 3x(x2 − 4).
  2. x2 − 4 is a difference of squares: (x + 2)(x − 2).

Answer: 3x(x + 2)(x − 2)

Problem 7.8

Factor: x2 + 14x + 49

Show solution
  1. Perfect square trinomial: a = x, b = 7 (since 2·7 = 14 and 72 = 49).

Answer: (x + 7)2

Chapter 8 — Quadratic Functions & Equations

Standard form y = ax2 + bx + c. Vertex form y = a(x − h)2 + k. Axis of symmetry x = −b/(2a). Discriminant D = b2 − 4ac. To solve quadratics: factor, square roots, complete the square, or quadratic formula. For deeper Ch 8 practice see the dedicated Ch 8 Workbook.

Problem 8.1

Identify the vertex and axis of symmetry of y = (x − 4)2 + 7.

Show solution
  1. Vertex form y = a(x − h)2 + k → vertex (4, 7); axis x = 4.

Answer: Vertex (4, 7); axis x = 4

Problem 8.2

Solve by factoring: x2 − 7x + 12 = 0

Show solution
  1. Factor: (x − 3)(x − 4) = 0.
  2. Set each factor = 0.

Answer: x = 3, x = 4

Problem 8.3

Solve by quadratic formula: 3x2 + 5x − 2 = 0

Show solution
  1. a = 3, b = 5, c = −2.
  2. D = 52 − 4(3)(−2) = 25 + 24 = 49. √49 = 7.
  3. x = (−5 ± 7)/6.
  4. x = 2/6 = 1/3, OR x = −12/6 = −2.

Answer: x = 1/3, x = −2

Problem 8.4

Find the discriminant of y = 4x2 − 12x + 9 and number of real solutions.

Show solution
  1. D = (−12)2 − 4(4)(9) = 144 − 144 = 0.

Answer: D = 0; one (double) real solution.

Problem 8.5

A rocket is launched from a 6-foot-high platform at 64 ft/s. Its height is h(t) = −16t2 + 64t + 6. What is the maximum height?

Show solution
  1. Max at t = −b/(2a) = −64/(2·(−16)) = 2 sec.
  2. h(2) = −16(4) + 64(2) + 6 = −64 + 128 + 6 = 70.

Answer: Max height = 70 ft

Problem 8.6

Convert to vertex form by completing the square: y = x2 − 6x + 11

Show solution
  1. Group: y = (x2 − 6x) + 11.
  2. Half of −6 is −3; (−3)2 = 9. Add and subtract 9 inside: y = (x2 − 6x + 9) + 11 − 9.
  3. y = (x − 3)2 + 2.

Answer: y = (x − 3)2 + 2; vertex (3, 2)

Chapter 9 — Radicals, Pythagorean & Distance

Simplify radicals by factoring out perfect squares. Rationalize denominators by multiplying by the radical. Pythagorean: a2 + b2 = c2 (right triangles). Distance formula: d = √[(x2−x1)2 + (y2−y1)2].

Problem 9.1

Simplify: √72

Show solution
  1. Factor out perfect squares: 72 = 36 · 2.
  2. √72 = √36 · √2 = 6√2.

Answer: 6√2

Problem 9.2

Simplify: √20 + √45

Show solution
  1. √20 = 2√5; √45 = 3√5.
  2. Add like radicals: 2√5 + 3√5 = 5√5.

Answer: 5√5

Problem 9.3

Rationalize: 4 / √3

Show solution
  1. Multiply numerator and denominator by √3.
  2. = 4√3 / 3.

Answer: 4√3 / 3

Problem 9.4

A right triangle has legs 8 and 15. Find the hypotenuse.

Show solution
  1. c2 = 82 + 152 = 64 + 225 = 289.
  2. c = √289 = 17.

Answer: Hypotenuse = 17

Problem 9.5

Find the distance between (1, −3) and (5, 9).

Show solution
  1. d = √[(5−1)2 + (9−(−3))2]
  2. = √[42 + 122] = √[16 + 144] = √160.
  3. Simplify: √160 = √(16·10) = 4√10.

Answer: d = 4√10 (≈ 12.65)

Problem 9.6

Find the midpoint of (−4, 7) and (10, −1).

Show solution
  1. M = ((−4 + 10)/2, (7 + (−1))/2) = (6/2, 6/2) = (3, 3).

Answer: Midpoint (3, 3)
